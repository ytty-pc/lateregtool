🎮 PCゲーム遅延最適化ツール
GPUとOSを選ぶだけで、あなたの環境に合った遅延削減設定と実行コマンドを一覧表示するWebツールです。
👉 ツールを使ってみる

概要
PCゲームにおける遅延（レイテンシ）は、マウスを動かしてから画面に反映されるまでの時間です。この時間はOS・ドライバー・GPU・プロセス設定など複数のレイヤーで積み重なります。
このツールでは、環境を選択するだけで有効な設定と手順・実行コマンドを自動で絞り込み表示します。

対応環境
項目対応内容GPUNVIDIA（GeForce）/ AMD（Radeon）OSWindows 10 / Windows 11ブラウザChrome / Edge / Firefox（最新版推奨）

主な機能

環境選択（GPU・OS・ゲームジャンル）による設定の自動絞り込み
各設定の手順・PowerShellコマンドの表示
コマンドのワンクリックコピー
.batファイルの自動生成・ダウンロード
チェックリストによる進捗管理


収録設定項目

NVIDIA Reflex / AMD Anti-Lag の有効化
Ultra Low Latency Mode（Reflex非対応ゲーム向け）
HAGS（ハードウェアアクセラレートGPUスケジューリング）
電源プランの最適化
タイマー解像度の1ms化
NIC割り込み節約機能の無効化
ゲームプロセス優先度の設定
GPUスケジューラ優先度のレジストリ変更
